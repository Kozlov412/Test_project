Обоснование выбора индексов и полей:

1. Одиночные индексы:

1.1 Order.status (db_index=True):
   - Ускоряет запросы при фильтрации заказов по статусу (например, "подтвержденные", "не подтвержденные")
   - Улучшает производительность при поиске заказов определенного статуса
   - Востребован в представлении orders_list, где мы фильтруем заказы по статусам

1.2 Order.date_created (db_index=True):
   - Используется для сортировки заказов по дате создания
   - Ускоряет запросы типа Order.objects.order_by('-date_created')
   - Повышает скорость выборки последних заказов на главной странице

1.3 Master.name (db_index=True):
   - Ускоряет поиск мастеров по имени
   - Используется при сортировке списка мастеров по алфавиту
   - Улучшает производительность автозаполнения и поиска

1.4 Master.is_active (db_index=True):
   - Часто используется в фильтрах для отображения только активных мастеров
   - Ускоряет запросы Master.objects.filter(is_active=True)
   - Важен для главной страницы, где отображаются только активные мастера

1.5 Service.name (db_index=True):
   - Используется для поиска услуг по названию
   - Востребован при алфавитной сортировке услуг
   - Улучшает скорость автозаполнения и поиска

1.6 Service.is_popular (db_index=True):
   - Используется для фильтрации популярных услуг на главной странице
   - Ускоряет запросы Service.objects.filter(is_popular=True)
   - Позволяет быстро выделить топовые услуги для рекламных блоков

2. Составные индексы:

2.1 Order: Index(fields=['status', 'appointment_date']):
   - Ускоряет частые запросы по поиску заказов определенного статуса на конкретную дату
   - Например, "все подтвержденные заказы на сегодня/завтра"
   - Помогает оптимизировать расписание мастеров, отображая заказы определенного статуса на выбранный день

2.2 Master: Index(fields=['experience', 'is_active']):
   - Используется при фильтрации активных мастеров с определенным стажем
   - Полезен для запросов типа "найти активных мастеров со стажем более 5 лет"
   - Помогает клиентам выбрать опытного мастера из числа активных

2.3 Service: Index(fields=['price', 'is_popular']):
   - Оптимизирует сортировку популярных услуг по цене
   - Ускоряет запросы типа "найти популярные услуги дешевле определенной цены"
   - Востребован при создании акций и спецпредложений

Эти индексы выбраны с учетом наиболее частых запросов, которые выполняются в приложении барбершопа, и будут способствовать значительному повышению производительности.
